Graphes

Définitions, représentation, parcours en largeur et en profondeur

I. Qu'est-ce qu'un graphe ?

Définition

Un graphe est un ensemble de sommets reliés par des arêtes (graphe non orienté) ou des arcs (graphe orienté).

  • Graphe non orienté : les arêtes n'ont pas de sens (une route à double sens).
  • Graphe orienté : les arcs ont un sens (une rue à sens unique).
  • Graphe pondéré : chaque arête a un poids (la distance entre deux villes).
Où trouve-t-on des graphes ?

Les graphes sont partout : réseau social (sommets = personnes, arêtes = amitié), réseau routier (sommets = villes, arêtes = routes), Internet (sommets = sites, arcs = liens hypertexte), réseau électrique, structure d'une molécule...

II. Représentation en Python

1. Matrice d'adjacence

# Matrice d'adjacence : une grille n × n
# matrice[i][j] = 1 si i et j sont reliés, 0 sinon

matrice = [
    #  A  B  C  D  E
    [0, 1, 1, 0, 0],   # A
    [1, 0, 1, 1, 0],   # B
    [1, 1, 0, 0, 1],   # C
    [0, 1, 0, 0, 1],   # D
    [0, 0, 1, 1, 0],   # E
]

# A est relié à B ?
print(matrice[0][1])   # 1 → oui

# Nombre de voisins de B (degré de B) :
print(sum(matrice[1]))   # 3

2. Dictionnaire d'adjacence (liste de voisins)

# Plus pratique pour la plupart des algorithmes

graphe = {
    "A": ["B", "C"],
    "B": ["A", "C", "D"],
    "C": ["A", "B", "E"],
    "D": ["B", "E"],
    "E": ["C", "D"],
}

# Voisins de B :
print(graphe["B"])     # ['A', 'C', 'D']

# A est-il relié à D ?
print("D" in graphe["A"])   # False

# Degré de B :
print(len(graphe["B"]))     # 3

La case (i, j) est colorée si le sommet i est relié au sommet j.

III. Parcours de graphes

1. Parcours en largeur (BFS : Breadth First Search)

Principe

On explore le graphe couche par couche, en visitant d'abord tous les voisins directs, puis les voisins des voisins, etc. On utilise une file (FIFO).

from collections import deque

def parcours_largeur(graphe, depart):
    """
    Parcourt le graphe en largeur depuis 'depart'.
    Retourne la liste des sommets visités dans l'ordre.
    """
    visites = []
    file = deque([depart])
    vus = {depart}

    while file:
        sommet = file.popleft()       # défiler
        visites.append(sommet)

        for voisin in graphe[sommet]:
            if voisin not in vus:
                vus.add(voisin)
                file.append(voisin)   # enfiler

    return visites


print(parcours_largeur(graphe, "A"))
# ['A', 'B', 'C', 'D', 'E']

Visualiser : parcours en largeur (BFS)

Le BFS explore couche par couche, comme des cercles concentriques.

2. Parcours en profondeur (DFS : Depth First Search)

Principe

On explore le graphe en allant le plus loin possible dans une direction avant de revenir en arrière. On utilise une pile (LIFO) ou la récursivité.

def parcours_profondeur(graphe, depart):
    """
    Parcourt le graphe en profondeur depuis 'depart'.
    Version récursive.
    """
    visites = []

    def dfs(sommet, vus):
        vus.add(sommet)
        visites.append(sommet)

        for voisin in graphe[sommet]:
            if voisin not in vus:
                dfs(voisin, vus)

    dfs(depart, set())
    return visites


print(parcours_profondeur(graphe, "A"))
# ['A', 'B', 'C', 'E', 'D']

Visualiser : parcours en profondeur (DFS)

Le DFS va le plus loin possible avant de revenir en arrière (backtracking).

Corrigé

1. BFS depuis A : A → B, C (voisins de A) → D (voisin de B non visité) → E (voisin de C non visité). Ordre : [A, B, C, D, E].

2. DFS depuis A : A → B → D → C → E. On va le plus loin possible dans chaque branche.

def chemin_existe(graphe, depart, arrivee):
    """
    Retourne True si un chemin existe entre
    depart et arrivee dans le graphe.
    """
    vus = set()

    def dfs(sommet):
        if sommet == arrivee:
            return True

        vus.add(sommet)

        for voisin in graphe[sommet]:
            if voisin not in vus:
                if dfs(voisin):
                    return True

        return False

    return dfs(depart)


print(chemin_existe(graphe, "A", "E"))   # True
print(chemin_existe(graphe, "A", "Z"))   # KeyError ou False
