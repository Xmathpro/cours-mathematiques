Trigonométrie

Cercle trigonométrique, radians, cosinus, sinus et équations

I. Le cercle trigonométrique

Définition

Le cercle trigonométrique est le cercle de centre $O$ et de rayon 1, orienté dans le sens inverse des aiguilles d'une montre (sens positif).

À chaque réel $x$ on associe un unique point $M$ sur le cercle, obtenu en "enroulant" la droite des réels autour du cercle à partir du point $(1, 0)$.

Radian

Le radian est l'unité de mesure des angles en mathématiques. Un tour complet correspond à $2\pi$ radians.

II. Cosinus et sinus

Définition

Soit $M$ le point du cercle trigonométrique associé au réel $x$. Les coordonnées de $M$ sont $(\cos x,\; \sin x)$.

  • $\cos x$ est l'abscisse de $M$ (projection sur l'axe horizontal)
  • $\sin x$ est l'ordonnée de $M$ (projection sur l'axe vertical)
Valeurs remarquables
$x$ $0$$\dfrac{\pi}{6}$$\dfrac{\pi}{4}$ $\dfrac{\pi}{3}$$\dfrac{\pi}{2}$$\pi$
$\cos x$ $1$$\dfrac{\sqrt{3}}{2}$$\dfrac{\sqrt{2}}{2}$ $\dfrac{1}{2}$$0$$-1$
$\sin x$ $0$$\dfrac{1}{2}$$\dfrac{\sqrt{2}}{2}$ $\dfrac{\sqrt{3}}{2}$$1$$0$

III. Propriétés fondamentales

Propriétés
  • Relation fondamentale : $\cos^2 x + \sin^2 x = 1$ pour tout $x$
  • $-1 \leq \cos x \leq 1$ et $-1 \leq \sin x \leq 1$
  • Périodicité : $\cos(x + 2\pi) = \cos x$ et $\sin(x + 2\pi) = \sin x$
  • Parité : $\cos(-x) = \cos x$ (paire) et $\sin(-x) = -\sin x$ (impaire)
Formules de symétrie
  • $\cos(\pi - x) = -\cos x$ et $\sin(\pi - x) = \sin x$
  • $\cos(\pi + x) = -\cos x$ et $\sin(\pi + x) = -\sin x$
  • $\cos\left(\dfrac{\pi}{2} - x\right) = \sin x$ et $\sin\left(\dfrac{\pi}{2} - x\right) = \cos x$
Formules d'addition
  • $\cos(a + b) = \cos a \cos b - \sin a \sin b$
  • $\cos(a - b) = \cos a \cos b + \sin a \sin b$
  • $\sin(a + b) = \sin a \cos b + \cos a \sin b$
  • $\sin(a - b) = \sin a \cos b - \cos a \sin b$

En particulier (formules de duplication) :

  • $\cos(2a) = \cos^2 a - \sin^2 a = 2\cos^2 a - 1 = 1 - 2\sin^2 a$
  • $\sin(2a) = 2\sin a \cos a$

IV. Équations trigonométriques

Méthode : résoudre $\cos x = \cos a$

$\cos x = \cos a \iff x = a + 2k\pi$ ou $x = -a + 2k\pi$ ($k \in \mathbb{Z}$).

Exemple. Résoudre $\cos x = \dfrac{1}{2}$ sur $[0, 2\pi[$.

$\cos x = \cos\dfrac{\pi}{3}$, donc $x = \dfrac{\pi}{3}$ ou $x = -\dfrac{\pi}{3} + 2\pi = \dfrac{5\pi}{3}$.

Méthode : résoudre $\sin x = \sin a$

$\sin x = \sin a \iff x = a + 2k\pi$ ou $x = \pi - a + 2k\pi$ ($k \in \mathbb{Z}$).

Exemple. Résoudre $\sin x = \dfrac{\sqrt{2}}{2}$ sur $[0, 2\pi[$.

$\sin x = \sin\dfrac{\pi}{4}$, donc $x = \dfrac{\pi}{4}$ ou $x = \pi - \dfrac{\pi}{4} = \dfrac{3\pi}{4}$.

V. Exercices

Exercice 1 : valeurs

Calculer les valeurs exactes de :

  1. $\cos\dfrac{5\pi}{6}$ et $\sin\dfrac{5\pi}{6}$
  2. $\cos\dfrac{7\pi}{4}$ et $\sin\dfrac{7\pi}{4}$
  3. $\cos\dfrac{2\pi}{3}$ et $\sin\dfrac{2\pi}{3}$

Exercice 2 : équations

Résoudre sur $[0, 2\pi[$ :

  1. $\cos x = -\dfrac{\sqrt{3}}{2}$
  2. $\sin x = -\dfrac{1}{2}$
  3. $2\cos^2 x - 1 = 0$
  4. $\sin(2x) = 0$ sur $[0, \pi[$

Exercice 3 : formules

  1. Calculer $\cos\dfrac{\pi}{12}$ en utilisant $\dfrac{\pi}{12} = \dfrac{\pi}{3} - \dfrac{\pi}{4}$.
  2. Simplifier $\cos^2 x - \sin^2 x + 2\sin x \cos x$.

VI. Corrigés

Corrigé exercice 1

Corrigé

1. $\dfrac{5\pi}{6} = \pi - \dfrac{\pi}{6}$.

$\cos\dfrac{5\pi}{6} = -\cos\dfrac{\pi}{6} = -\dfrac{\sqrt{3}}{2}$ et $\sin\dfrac{5\pi}{6} = \sin\dfrac{\pi}{6} = \dfrac{1}{2}$.

2. $\dfrac{7\pi}{4} = 2\pi - \dfrac{\pi}{4} = -\dfrac{\pi}{4} + 2\pi$.

$\cos\dfrac{7\pi}{4} = \cos\dfrac{\pi}{4} = \dfrac{\sqrt{2}}{2}$ et $\sin\dfrac{7\pi}{4} = -\sin\dfrac{\pi}{4} = -\dfrac{\sqrt{2}}{2}$.

3. $\dfrac{2\pi}{3} = \pi - \dfrac{\pi}{3}$.

$\cos\dfrac{2\pi}{3} = -\cos\dfrac{\pi}{3} = -\dfrac{1}{2}$ et $\sin\dfrac{2\pi}{3} = \sin\dfrac{\pi}{3} = \dfrac{\sqrt{3}}{2}$.

Corrigé exercice 2

Corrigé

1. $\cos x = -\dfrac{\sqrt{3}}{2} = \cos\dfrac{5\pi}{6}$.

$x = \dfrac{5\pi}{6}$ ou $x = -\dfrac{5\pi}{6} + 2\pi = \dfrac{7\pi}{6}$.

2. $\sin x = -\dfrac{1}{2} = \sin\left(-\dfrac{\pi}{6}\right)$.

$x = -\dfrac{\pi}{6} + 2\pi = \dfrac{11\pi}{6}$ ou $x = \pi + \dfrac{\pi}{6} = \dfrac{7\pi}{6}$.

3. $2\cos^2 x - 1 = 0 \iff \cos^2 x = \dfrac{1}{2} \iff \cos x = \pm\dfrac{\sqrt{2}}{2}$.

$x = \dfrac{\pi}{4}$ ou $x = \dfrac{3\pi}{4}$ ou $x = \dfrac{5\pi}{4}$ ou $x = \dfrac{7\pi}{4}$.

4. $\sin(2x) = 0 \iff 2x = k\pi \iff x = \dfrac{k\pi}{2}$ ($k \in \mathbb{Z}$).

Sur $[0, \pi[$ : $x = 0$ ou $x = \dfrac{\pi}{2}$.

Corrigé exercice 3

Corrigé

1. $\cos\dfrac{\pi}{12} = \cos\left(\dfrac{\pi}{3} - \dfrac{\pi}{4}\right) = \cos\dfrac{\pi}{3}\cos\dfrac{\pi}{4} + \sin\dfrac{\pi}{3}\sin\dfrac{\pi}{4}$

$= \dfrac{1}{2} \times \dfrac{\sqrt{2}}{2} + \dfrac{\sqrt{3}}{2} \times \dfrac{\sqrt{2}}{2} = \dfrac{\sqrt{2}}{4} + \dfrac{\sqrt{6}}{4} = \dfrac{\sqrt{2} + \sqrt{6}}{4}$.

2. $\cos^2 x - \sin^2 x + 2\sin x \cos x = \cos(2x) + \sin(2x)$.
