I. Primitives

Définition

$F$ est une primitive de $f$ sur $I$ si $F' = f$ sur $I$. Si $F$ est une primitive de $f$, alors toutes les primitives sont de la forme $F + c$ ($c \in \mathbb{R}$).

Primitives usuelles
$f(x)$$F(x)$
$x^n$ ($n \neq -1$)$\dfrac{x^{n+1}}{n+1}$
$\dfrac{1}{x}$$\ln|x|$
$e^x$$e^x$
$u' e^u$$e^u$
$\dfrac{u'}{u}$$\ln|u|$

II. Intégrale

Définition

$$\int_a^b f(x)\, dx = F(b) - F(a)$$

où $F$ est une primitive quelconque de $f$.

Propriétés
  • Linéarité : $\int_a^b (\lambda f + \mu g) = \lambda \int_a^b f + \mu \int_a^b g$
  • Chasles : $\int_a^b f = \int_a^c f + \int_c^b f$
  • Positivité : si $f \geq 0$ sur $[a,b]$ alors $\int_a^b f \geq 0$
  • Sens : $\int_a^b f = -\int_b^a f$

III. Intégration par parties

Formule
$$\int_a^b u(x)v'(x)\,dx = [u(x)v(x)]_a^b - \int_a^b u'(x)v(x)\,dx$$
Exemple

Calculer $\int_0^1 xe^x\,dx$. On pose $u = x$, $v' = e^x$. Donc $u' = 1$, $v = e^x$.

$\int_0^1 xe^x\,dx = [xe^x]_0^1 - \int_0^1 e^x\,dx = e - [e^x]_0^1 = e - (e-1) = 1$.

IV. Valeur moyenne

Définition

La valeur moyenne de $f$ sur $[a, b]$ est :

$$\mu = \frac{1}{b-a}\int_a^b f(x)\,dx$$

V. Python et intégration

Méthode des rectangles en Python
def integrale_rectangles(f, a, b, n=1000):
    h = (b - a) / n
    s = 0
    for i in range(n):
        s += f(a + i * h) * h
    return s

import math
# Calculer l'intégrale de e^x de 0 à 1
result = integrale_rectangles(math.exp, 0, 1)
print(f"Intégrale approchée : {result:.6f}")
print(f"Valeur exacte (e-1) : {math.e - 1:.6f}")

VI. Exercices

Exercice 1 : primitives et intégrales

  1. $\int_0^3 (2x+1)\,dx$
  2. $\int_1^e \dfrac{1}{x}\,dx$
  3. $\int_0^1 e^{2x}\,dx$
  4. $\int_0^{\pi} \sin x\,dx$

Exercice 2 : intégration par parties

  1. $\int_0^1 (x+1)e^x\,dx$
  2. $\int_1^e x\ln x\,dx$

Exercice 3 : aire

Calculer l'aire entre la courbe de $f(x) = x^2$ et la droite $y = 4$ sur $[-2, 2]$.

VII. Corrigés

Corrigé exercice 1

Corrigé

1. $[x^2 + x]_0^3 = (9+3) - 0 = 12$.

2. $[\ln x]_1^e = \ln e - \ln 1 = 1 - 0 = 1$.

3. $[\frac{1}{2}e^{2x}]_0^1 = \frac{1}{2}(e^2 - 1) \approx 3{,}19$.

4. $[-\cos x]_0^{\pi} = -\cos\pi + \cos 0 = 1 + 1 = 2$.

Corrigé exercice 2

Corrigé

1. $u = x+1$, $v' = e^x$. $u' = 1$, $v = e^x$.

$[(x+1)e^x]_0^1 - \int_0^1 e^x\,dx = 2e - 1 - (e-1) = e = e \approx 2{,}718$.

2. $u = \ln x$, $v' = x$. $u' = 1/x$, $v = x^2/2$.

$[\frac{x^2}{2}\ln x]_1^e - \int_1^e \frac{x}{2}\,dx = \frac{e^2}{2} - [\frac{x^2}{4}]_1^e = \frac{e^2}{2} - \frac{e^2-1}{4} = \frac{e^2+1}{4} \approx 2{,}10$.

Corrigé exercice 3

Corrigé

Sur $[-2, 2]$, $4 \geq x^2$. Aire $= \int_{-2}^{2} (4 - x^2)\,dx = [4x - \frac{x^3}{3}]_{-2}^{2}$.

$= (8 - \frac{8}{3}) - (-8 + \frac{8}{3}) = 16 - \frac{16}{3} = \frac{32}{3} \approx 10{,}67$ u.a.
