Exemples

1. $f(x) = e^{3x+1}$. $u = 3x+1$, $u' = 3$. $f'(x) = 3e^{3x+1}$.

2. $g(x) = \ln(x^2+1)$. $u = x^2+1$, $u' = 2x$. $g'(x) = \dfrac{2x}{x^2+1}$.

3. $h(x) = (2x-5)e^{-x}$. Produit : $h'(x) = 2e^{-x} + (2x-5)(-e^{-x}) = (2 - 2x + 5)e^{-x} = (7-2x)e^{-x}$.

II. Convexité

Définition

Soit $f$ deux fois dérivable sur un intervalle $I$.

  • $f$ est convexe sur $I$ si $f''(x) \geq 0$ sur $I$. La courbe est au-dessus de ses tangentes. Forme "creux vers le haut" $\cup$.
  • $f$ est concave sur $I$ si $f''(x) \leq 0$ sur $I$. La courbe est au-dessous de ses tangentes. Forme "creux vers le bas" $\cap$.
Point d'inflexion

Un point d'inflexion est un point où la courbe change de convexité (de convexe à concave ou inversement). En ce point, $f''(x_0) = 0$ et $f''$ change de signe.

III. Optimisation

Méthode d'optimisation

1. Modéliser le problème : identifier la variable et la fonction à optimiser.

2. Calculer la dérivée et résoudre $f'(x) = 0$.

3. Dresser le tableau de variations pour identifier le maximum ou minimum.

4. Conclure avec les valeurs optimales.

Exemple type bac

On modélise la concentration d'un médicament dans le sang par $C(t) = t \cdot e^{-0{,}5t}$ (en mg/L) où $t$ est le temps en heures.

Dérivée : $C'(t) = e^{-0{,}5t} + t(-0{,}5)e^{-0{,}5t} = (1-0{,}5t)e^{-0{,}5t}$.

Signe : $e^{-0{,}5t} > 0$ toujours, donc $C'(t)$ a le signe de $(1 - 0{,}5t)$.

$C'(t) = 0 \iff t = 2$. $C'(t) > 0$ sur $]0, 2[$ et $C'(t) < 0$ sur $]2, +\infty[$.

Maximum : $C(2) = 2e^{-1} \approx 0{,}74$ mg/L atteint à $t = 2$ heures.

À méditer

La concentration maximale d'un médicament et le temps pour l'atteindre sont des données cliniques essentielles. Elles dépendent de la molécule, de la dose, de la voie d'administration et du métabolisme du patient. Le modèle $C(t) = te^{-kt}$ est un modèle simplifié. En réalité, les laboratoires utilisent des modèles plus complexes calibrés sur des essais cliniques. Comment ces essais sont-ils conçus, financés et publiés est un sujet qui mérite réflexion.

Soit $f(x) = x^3 - 6x^2 + 9x + 1$.

Exercice 3 : optimisation

Le bénéfice d'une entreprise en fonction du prix $p$ de vente est $B(p) = -2p^2 + 120p - 1000$ (en euros).

  1. Pour quel prix le bénéfice est-il maximal ?
  2. Quel est le bénéfice maximal ?
  3. Pour quels prix l'entreprise fait-elle un bénéfice positif ?

VI. Corrigés

Corrigé exercice 1

Corrigé

1. $u = 2x^2 - x$, $u' = 4x - 1$. $f'(x) = (4x-1)e^{2x^2-x}$.

2. $u = 3x+7$, $u' = 3$. $g'(x) = \dfrac{3}{3x+7}$.

3. Quotient : $h'(x) = \dfrac{e^x(x+1) - e^x}{(x+1)^2} = \dfrac{xe^x}{(x+1)^2}$.

4. Produit : $k'(x) = 2x\ln x + x^2 \cdot \dfrac{1}{x} = 2x\ln x + x = x(2\ln x + 1)$.

Corrigé exercice 2

Corrigé

1. $f'(x) = 3x^2 - 12x + 9$. $f''(x) = 6x - 12$.

2. $f''(x) = 0 \iff x = 2$.

$f''(x) < 0$ sur $]-\infty, 2[$ : $f$ concave. $f''(x) > 0$ sur $]2, +\infty[$ : $f$ convexe.

3. Point d'inflexion en $x = 2$ : $f(2) = 8 - 24 + 18 + 1 = 3$. Le point est $(2, 3)$.

Corrigé exercice 3

Corrigé

1. $B'(p) = -4p + 120 = 0 \iff p = 30$ euros.

$B''(p) = -4 < 0$ : c'est bien un maximum.

2. $B(30) = -2(900) + 3600 - 1000 = -1800 + 3600 - 1000 = 800$ euros.

3. $B(p) > 0 \iff -2p^2 + 120p - 1000 > 0$. $\Delta = 14400 - 8000 = 6400$.

$p = \dfrac{-120 \pm 80}{-4}$. $p_1 = 10$ et $p_2 = 50$.

$B(p) > 0$ sur $]10, 50[$. L'entreprise fait du bénéfice pour un prix entre 10 et 50 euros.
