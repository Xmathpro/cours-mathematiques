II. Fonctions de référence

1. Fonction carré

$f(x) = x^2$
  • Définie sur $\mathbb{R}$. Ensemble image : $[0, +\infty[$.
  • Décroissante sur $]-\infty, 0]$, croissante sur $[0, +\infty[$.
  • Minimum : $f(0) = 0$.
  • Parité : $f(-x) = f(x)$ (symétrie par rapport à l'axe des ordonnées).

2. Fonction cube

$f(x) = x^3$
  • Définie sur $\mathbb{R}$. Ensemble image : $\mathbb{R}$.
  • Strictement croissante sur $\mathbb{R}$.
  • Imparité : $f(-x) = -f(x)$ (symétrie par rapport à l'origine).

3. Fonction inverse

$f(x) = \dfrac{1}{x}$
  • Définie sur $\mathbb{R} \setminus \{0\}$.
  • Strictement décroissante sur $]-\infty, 0[$ et sur $]0, +\infty[$.
  • Attention : on ne dit PAS "décroissante sur $\mathbb{R}^*$" car $f(-1) = -1 < 1 = f(1)$.
  • Imparité : $f(-x) = -f(x)$.

4. Fonction racine carrée

$f(x) = \sqrt{x}$
  • Définie sur $[0, +\infty[$.
  • Strictement croissante sur $[0, +\infty[$.
  • La croissance ralentit : la courbe "s'aplatit" progressivement.

III. Résolution d'équations et d'inéquations

Méthode : résoudre $f(x) = g(x)$

On ramène à $f(x) - g(x) = 0$ et on factorise ou on utilise le discriminant.

Méthode : résoudre une inéquation avec un tableau de signes

Exemple. Résoudre $(2x-1)(x+3) \leq 0$.

Les racines sont $x = \dfrac{1}{2}$ et $x = -3$.

  • Sur $]-\infty, -3[$ : $(2x-1) < 0$ et $(x+3) < 0$, produit $> 0$.
  • Sur $]-3, \dfrac{1}{2}[$ : $(2x-1) < 0$ et $(x+3) > 0$, produit $< 0$.
  • Sur $]\dfrac{1}{2}, +\infty[$ : $(2x-1) > 0$ et $(x+3) > 0$, produit $> 0$.

Solution : $x \in [-3, \dfrac{1}{2}]$.

IV. Algorithmique et Python

Nouveauté du programme 2019

Python est désormais l'outil officiel pour explorer les fonctions en classe de Seconde. On l'utilise pour calculer des images, tracer des courbes et résoudre des problèmes par balayage.

1. Calculer des images avec Python

Exemple
def f(x):
    return x**2 - 3*x + 2

# Calculer des images
print(f(0))    # affiche 2
print(f(1))    # affiche 0
print(f(5))    # affiche 12

# Table de valeurs
for x in range(-5, 6):
    print("f(", x, ") =", f(x))

2. Tracer une courbe avec Python

Exemple
import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(-5, 5, 200)
y = x**2 - 3*x + 2

plt.plot(x, y, color='blue')
plt.axhline(y=0, color='gray', linewidth=0.5)
plt.axvline(x=0, color='gray', linewidth=0.5)
plt.grid(True, alpha=0.3)
plt.title("f(x) = x² - 3x + 2")
plt.show()

3. Résolution par balayage

Principe du balayage

Pour trouver une solution approchée de $f(x) = 0$, on parcourt un intervalle en testant les valeurs de $f$ pas à pas. Quand $f$ change de signe entre deux valeurs consécutives, la solution est entre les deux.

Exemple : trouver $\sqrt{2}$ par balayage
# On cherche x tel que x² = 2, soit x² - 2 = 0
def f(x):
    return x**2 - 2

# Balayage entre 1 et 2 avec un pas de 0.001
x = 1.0
while f(x) < 0:
    x = x + 0.001

print("sqrt(2) ≈", round(x, 3))
# Affiche : sqrt(2) ≈ 1.415

V. Exercices

Exercice 1 : fonctions de référence

  1. Comparer sans calculatrice : $\sqrt{5}$ et $\sqrt{7}$. Justifier en utilisant la monotonie de la fonction racine.
  2. Comparer $(-3)^2$ et $(-5)^2$. Justifier en utilisant la monotonie de la fonction carré.
  3. Comparer $\dfrac{1}{3}$ et $\dfrac{1}{7}$. Justifier en utilisant la monotonie de la fonction inverse.

Exercice 2 : inéquations

Résoudre :

  1. $(x-2)(x+5) > 0$
  2. $\dfrac{x+1}{x-3} \leq 0$
  3. $x^2 - 4x + 3 < 0$

Exercice 3 : Python

Écrire un programme Python qui :

  1. Affiche la table de valeurs de $f(x) = x^2 - 5x + 4$ pour $x$ allant de $-2$ à $7$.
  2. Trouve par balayage les solutions de $f(x) = 0$ avec une précision de $0{,}01$.

VI. Corrigés

Corrigé exercice 1

Corrigé

1. $5 < 7$ et la fonction racine est croissante sur $[0, +\infty[$, donc $\sqrt{5} < \sqrt{7}$.

2. On compare $|-3| = 3$ et $|-5| = 5$. Comme $3 < 5$ et que la fonction carré est croissante sur $[0, +\infty[$ : $3^2 < 5^2$, soit $(-3)^2 < (-5)^2$, soit $9 < 25$.

3. $3 < 7$ et la fonction inverse est décroissante sur $]0, +\infty[$, donc $\dfrac{1}{3} > \dfrac{1}{7}$.

Corrigé exercice 2

Corrigé

1. Racines : $x = 2$ et $x = -5$. Tableau de signes :

$(x-2)(x+5) > 0$ sur $]-\infty, -5[ \cup ]2, +\infty[$.

2. Racine du numérateur : $x = -1$. Valeur interdite : $x = 3$.

$\dfrac{x+1}{x-3} \leq 0$ sur $[-1, 3[$.

3. $x^2 - 4x + 3 = (x-1)(x-3)$. Racines : $x = 1$ et $x = 3$.

$(x-1)(x-3) < 0$ sur $]1, 3[$.

Corrigé exercice 3

Corrigé
# 1. Table de valeurs
def f(x):
    return x**2 - 5*x + 4

for x in range(-2, 8):
    print("f(", x, ") =", f(x))

# 2. Balayage
x = -2.0
while x <= 7:
    if abs(f(x)) < 0.01:
        print("Solution approchée :", round(x, 2))
    x += 0.01
# Trouve x ≈ 1.00 et x ≈ 4.00
# Vérification : f(x) = (x-1)(x-4), racines 1 et 4.
