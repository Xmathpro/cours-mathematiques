Le Web : HTML et CSS

Structure d'une page, balises HTML, mise en forme CSS et modèle client-serveur

I. Comment fonctionne le Web ?

Le modèle client-serveur

Quand tu tapes une adresse dans ton navigateur :

  1. Ton navigateur (le client) envoie une requête HTTP au serveur.
  2. Le serveur reçoit la requête et renvoie une réponse contenant le fichier HTML.
  3. Le navigateur lit le HTML, charge le CSS et le JavaScript, puis affiche la page.

Visualiser : le parcours d'une requête Web

Cliquez pour voir le parcours d'une requête Web.
Les 3 langages du Web
  • HTML : le contenu et la structure (le squelette).
  • CSS : la mise en forme et le design (l'apparence).
  • JavaScript : l'interactivité et le comportement (les actions).

II. HTML : la structure

Qu'est-ce que le HTML ?

HTML (HyperText Markup Language) est un langage de balisage : on entoure le contenu de balises pour lui donner un sens. Ce n'est pas un langage de programmation : il ne fait pas de calculs.

1. Structure minimale d'une page

<!DOCTYPE html>
<html lang="fr">

  <head>
    <meta charset="UTF-8">
    <title>Ma première page</title>
  </head>

  <body>
    <h1>Bienvenue !</h1>
    <p>Ceci est un paragraphe.</p>
  </body>

</html>

Visualiser : l'arbre HTML (DOM)

Le navigateur transforme le HTML en un arbre : le DOM (Document Object Model). Chaque balise est un noeud de l'arbre.

III. CSS : la mise en forme

Qu'est-ce que le CSS ?

CSS (Cascading Style Sheets) contrôle l'apparence : couleurs, tailles, polices, marges, disposition. On sépare le contenu (HTML) de la présentation (CSS).

1. Syntaxe CSS

/* Sélecteur { propriété: valeur; } */

h1 {
    color: blue;
    font-size: 32px;
    text-align: center;
}

p {
    color: #333333;
    font-family: Arial, sans-serif;
    line-height: 1.6;
}

.important {
    background-color: yellow;
    border: 2px solid red;
    padding: 10px;
}

2. Les sélecteurs

3 types de sélecteurs
Type Syntaxe Cible
Balisep { ... }Tous les paragraphes
Classe.important { ... }Éléments avec class="important"
Identifiant#titre { ... }L'élément avec id="titre"

IV. Exercices

Exercice 1 : lire du HTML

<h2>Recette</h2>
<p>Voici les <strong>ingrédients</strong> :</p>
<ul>
  <li>200g de farine</li>
  <li>3 oeufs</li>
  <li>50cl de lait</li>
</ul>
  1. Combien de balises ouvrantes y a-t-il ?
  2. Quel texte apparaîtra en gras ?
  3. Dessiner l'arbre DOM de ce fragment.

Exercice 2 : écrire du HTML

Écrire le code HTML d'une page contenant :

  1. Un titre principal "Mon CV".
  2. Un paragraphe de présentation avec votre prénom en gras.
  3. Une liste numérotée de 3 compétences.
  4. Un lien vers le site de votre école.

Exercice 3 : CSS

Écrire le CSS pour :

  1. Mettre tous les titres h1 en bleu foncé.
  2. Donner un fond gris clair à tous les paragraphes avec un arrondi de 5px.
  3. Créer une classe .alerte avec fond rouge, texte blanc et padding de 15px.

V. Corrigés

Corrigé exercice 1

Corrigé

1. 7 balises ouvrantes : <h2>, <p>, <strong>, <ul>, 3 <li>.

3. L'arbre : le fragment contient 3 noeuds au premier niveau (h2, p, ul). p contient du texte et strong. ul contient 3 li.
