IV. Boucles

1. Boucle for

La boucle for

La boucle for répète une instruction un nombre connu à l'avance de fois.

for i in range(5):
    print(i)  # affiche 0, 1, 2, 3, 4

range(n) génère les entiers de 0 à n-1. range(a, b) de a à b-1. range(a, b, pas) avec un pas.

2. Boucle while

La boucle while

La boucle while répète une instruction tant qu'une condition est vraie. On l'utilise quand on ne sait pas à l'avance combien de fois répéter.

x = 1
while x < 100:
    x = x * 2
print(x)  # affiche 128

V. Fonctions

Définir une fonction

Une fonction est un bloc de code réutilisable. On la définit avec def, on lui donne des paramètres et elle retourne un résultat.

def carre(x):
    return x ** 2

print(carre(5))   # affiche 25
print(carre(-3))  # affiche 9

VI. Structures de données : listes

La liste

Une liste est une séquence ordonnée d'éléments. On accède aux éléments par leur indice, qui commence à 0.

fruits = ["pomme", "banane", "cerise"]
print(fruits[0])   # pomme (premier élément)
print(fruits[-1])  # cerise (dernier élément)
fruits.append("datte")  # ajouter en fin

VII. Exercices

Exercice 1 : variables et types

  1. Quels sont les types de 42, 3.14, "NSI", True ?
  2. Que vaut 17 // 5 ? Et 17 % 5 ? Que représentent ces opérations ?
  3. Que vaut "NSI" + "2024" ? Et "ha" * 3 ?

Exercice 2 : conditions

Écrire une fonction classification(imc) qui retourne :

  • "Insuffisance pondérale" si IMC < 18.5
  • "Poids normal" si 18.5 ≤ IMC < 25
  • "Surpoids" si 25 ≤ IMC < 30
  • "Obésité" si IMC ≥ 30

Exercice 3 : boucles

  1. Écrire une fonction somme(n) qui calcule $1 + 2 + \cdots + n$.
  2. Vérifier avec la formule $n(n+1)/2$.
  3. Écrire une fonction factorielle(n) qui calcule $n!$.
  4. Trouver le plus petit entier $n$ tel que $n! > 10^6$.

Exercice 4 : listes

  1. Créer la liste des carrés de 1 à 10 avec une compréhension de liste.
  2. Écrire une fonction maximum(liste) sans utiliser la fonction max().
  3. Écrire une fonction inverser(liste) qui retourne la liste dans l'ordre inverse.

Corrigé exercice 2

Corrigé
def classification(imc):
    if imc < 18.5:
        return "Insuffisance pondérale"
    elif imc < 25:
        return "Poids normal"
    elif imc < 30:
        return "Surpoids"
    else:
        return "Obésité"

# Tests
print(classification(16))   # Insuffisance pondérale
print(classification(22))   # Poids normal
print(classification(27))   # Surpoids
print(classification(35))   # Obésité

Corrigé exercice 3

Corrigé
def somme(n):
    s = 0
    for i in range(1, n + 1):
        s += i
    return s

# Vérification
print(somme(10))        # 55
print(10 * 11 // 2)     # 55

def factorielle(n):
    f = 1
    for i in range(2, n + 1):
        f *= i
    return f

# Plus petit n tel que n! > 1 000 000
n = 1
while factorielle(n) <= 1000000:
    n += 1
print(n)  # 10 (car 10! = 3628800 > 1000000)

Corrigé exercice 4

Corrigé
# 1. Compréhension de liste
carres = [i**2 for i in range(1, 11)]
print(carres)  # [1, 4, 9, 16, 25, 36, 49, 64, 81, 100]

# 2. Maximum sans max()
def maximum(liste):
    if len(liste) == 0:
        return None
    maxi = liste[0]
    for val in liste:
        if val > maxi:
            maxi = val
    return maxi

# 3. Inverser
def inverser(liste):
    n = len(liste)
    return [liste[n - 1 - i] for i in range(n)]
# Ou plus simplement : return liste[::-1]
