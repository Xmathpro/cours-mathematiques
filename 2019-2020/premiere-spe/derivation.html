Dérivation

Nombre dérivé, tangente, dérivées usuelles, sens de variation et Python

I. Nombre dérivé et tangente

Nombre dérivé

$f'(a) = \lim_{h \to 0} \dfrac{f(a+h) - f(a)}{h}$

C'est le coefficient directeur de la tangente en $x = a$.

Équation de la tangente
$$y = f'(a)(x - a) + f(a)$$

Explorer : tangente mobile

1.0

II. Dérivées usuelles et règles de calcul

Dérivées usuelles

$(x^n)' = nx^{n-1}$, $(\sqrt{x})' = \dfrac{1}{2\sqrt{x}}$, $(1/x)' = -1/x^2$.

Opérations
  • $(u+v)' = u'+v'$
  • $(\lambda u)' = \lambda u'$
  • $(uv)' = u'v + uv'$

III. Sens de variation

Théorème
  • $f'(x) \geq 0$ sur $I$ $\iff$ $f$ croissante sur $I$
  • $f'(x) \leq 0$ sur $I$ $\iff$ $f$ décroissante sur $I$

IV. Python

Tracer la tangente en Python
import numpy as np
import matplotlib.pyplot as plt

def f(x): return x**3 - 3*x
def fp(x): return 3*x**2 - 3

x = np.linspace(-3, 3, 200)
x0 = 1
tang = fp(x0) * (x - x0) + f(x0)

plt.plot(x, f(x), 'b', label='f(x)')
plt.plot(x, tang, 'r--', label=f'Tangente en x={x0}')
plt.plot(x0, f(x0), 'ro')
plt.legend(); plt.grid(alpha=0.3); plt.ylim(-5, 5)
plt.show()

Exercice 2

$f(x) = -x^3 + 3x^2 + 9x - 5$. Variations et extremums.

VI. Corrigés

Corrigé exercice 1

Corrigé

1. $f'(x) = 12x^3 - 4x + 1$.

2. $g'(x) = 2(x-3) + (2x+1) = 4x - 5$.

3. $h'(x) = \dfrac{(x-2) - (x+1)}{(x-2)^2} = \dfrac{-3}{(x-2)^2}$.

Corrigé exercice 2

Corrigé

$f'(x) = -3x^2 + 6x + 9 = -3(x^2 - 2x - 3) = -3(x-3)(x+1)$.

$f'(x) = 0$ pour $x = -1$ et $x = 3$. $f'(x) > 0$ sur $]-1, 3[$.

$f(-1) = 1+3-9-5 = -10$ (min). $f(3) = -27+27+27-5 = 22$ (max).
