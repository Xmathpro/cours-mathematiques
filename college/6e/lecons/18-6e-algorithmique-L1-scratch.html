Scratch : initiation a la programmation

Sequences, boucles, conditions et deplacements

Algorithmique 6eme Lecon 1

I. Qu'est-ce que Scratch ?

Definition

Scratch est un langage de programmation visuel. On assemble des blocs colores comme des briques pour creer un programme. Chaque couleur correspond a une categorie :

  • Mouvement : deplacer le personnage (le "lutin")
  • Apparence : changer l'apparence, parler
  • Controle : boucles, conditions
  • Evenements : declencher le programme
  • Operateurs : calculs mathematiques

II. La sequence

Definition

Une sequence est une suite d'instructions executees l'une apres l'autre, dans l'ordre. C'est le principe de base de tout programme.

III. Les boucles

Repeter

La boucle repeter N fois execute les blocs a l'interieur N fois. Au lieu d'ecrire 10 fois la meme chose, on ecrit une seule fois dans une boucle.

IV. Les conditions

Si ... alors

La condition si ... alors permet au programme de prendre une decision. Si la condition est vraie, on execute les blocs a l'interieur. Sinon, on les saute (ou on execute un bloc "sinon").

V. Exercices

  1. Ecrire un programme Scratch qui trace un triangle equilateral (3 cotes, tourner de 120 degres).
  2. Ecrire un programme qui fait avancer le lutin de 100 pas, puis le fait tourner de 90 degres, et repete cela 4 fois. Quelle figure obtient-on ?
  3. Ecrire un programme avec une condition : si le lutin touche le bord, il rebondit.
  4. Combien de fois faut-il repeter "avancer de 50, tourner de 60 degres" pour tracer un hexagone ?
  5. Un programme repete 360 fois : "avancer de 1, tourner de 1 degre". Quelle figure obtient-on ?

VI. Corriges

Corriges

1. Repeter 3 fois : [avancer de 100, tourner de 120 degres].

2. Un carre de cote 100.

3. Si x > 240 ou x < -240 : tourner de 180 degres.

4. 6 fois (360 / 60 = 6 cotes).

5. Un cercle (ou presque). 360 petits pas avec 1 degre de rotation a chaque fois dessine un cercle.
