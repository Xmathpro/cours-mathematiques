Addition de fractions (denominateurs differents)

Reduire au meme denominateur pour additionner

I. Methode

Regle

Pour additionner deux fractions de denominateurs differents :

  1. On cherche un denominateur commun (en general, le produit des deux denominateurs, ou leur plus petit multiple commun).
  2. On transforme chaque fraction pour obtenir ce denominateur commun.
  3. On additionne les numerateurs.

$$\frac{a}{b} + \frac{c}{d} = \frac{a \times d}{b \times d} + \frac{c \times b}{d \times b} = \frac{a \times d + c \times b}{b \times d}$$

Exemple detaille

$\dfrac{2}{3} + \dfrac{3}{5}$

Denominateur commun : $3 \times 5 = 15$.

$\dfrac{2}{3} = \dfrac{2 \times 5}{3 \times 5} = \dfrac{10}{15}$ et $\dfrac{3}{5} = \dfrac{3 \times 3}{5 \times 3} = \dfrac{9}{15}$.

$\dfrac{10}{15} + \dfrac{9}{15} = \dfrac{19}{15}$.

II. Cas ou un denominateur est multiple de l'autre

Exemple

$\dfrac{1}{3} + \dfrac{5}{6}$ : comme $6 = 3 \times 2$, le denominateur commun est $6$.

$\dfrac{1}{3} = \dfrac{2}{6}$. Donc $\dfrac{2}{6} + \dfrac{5}{6} = \dfrac{7}{6}$.

Astuce

Quand un denominateur est un multiple de l'autre, il suffit de transformer une seule fraction. C'est plus rapide !

III. Soustraction

Meme methode

La soustraction suit exactement le meme principe : on reduit au meme denominateur puis on soustrait les numerateurs.

$$\frac{a}{b} - \frac{c}{d} = \frac{a \times d - c \times b}{b \times d}$$

Exemple

$\dfrac{5}{6} - \dfrac{1}{4} = \dfrac{5 \times 4}{6 \times 4} - \dfrac{1 \times 6}{4 \times 6} = \dfrac{20}{24} - \dfrac{6}{24} = \dfrac{14}{24} = \dfrac{7}{12}$
