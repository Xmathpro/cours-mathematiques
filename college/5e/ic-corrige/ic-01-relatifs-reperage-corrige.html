Corrige IC : Nombres relatifs : reperage

Question 1

Q : Placer les nombres $+4$, $-3$, $-5{,}5$, $+1$ sur une droite graduee.

R : On place chaque point a la bonne position sur la droite.

Question 2

Q : Donner l'oppose de : $+6$, $-2$, $+0{,}5$, $-9$.

R : $-6$, $+2$, $-0{,}5$, $+9$.

Question 3

Q : Ranger dans l'ordre croissant : $-4$, $+1$, $-7$, $+3$, $0$.

R : $-7 < -4 < 0 < +1 < +3$.

Question 4

Q : Donner les coordonnees du point $A$ d'abscisse $-2$ et d'ordonnee $5$.

R : $A(-2\,;\,5)$.
