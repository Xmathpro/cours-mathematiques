Corrige IC : Operations sur les relatifs

Question 1

Q : Calculer : $(+8)+(-3)$, $(-5)+(-7)$, $(+4)+(-4)$.

R : $+5$, $-12$, $0$.

Question 2

Q : Calculer : $(+6)-(+9)$, $(-3)-(-8)$, $(+2)-(-5)$.

R : $-3$, $+5$, $+7$.

Question 3

Q : Calculer : $(-2)+(+7)+(-5)+(+3)+(-1)$.

R : Positifs : $10$. Negatifs : $-8$. Total : $+2$.

Question 4

Q : La temperature passe de $-4$°C a $+7$°C. De combien a-t-elle varie ?

R : $(+7)-(-4)=(+7)+(+4)=+11$°C.
