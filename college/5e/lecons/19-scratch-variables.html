Scratch et variables

Variables, affectation, boucles, conditions

I. Qu'est-ce qu'une variable ?

Definition

En programmation, une variable est un espace de stockage qui porte un nom et qui contient une valeur. Cette valeur peut changer au cours du programme.

Exemple dans Scratch

On cree une variable appelee score et on lui donne la valeur 0 au debut du jeu. A chaque bonne reponse, on ajoute 1 a score.

II. Operations sur les variables

Blocs Scratch pour les variables

mettre score a 0 : donne la valeur 0 a la variable.

ajouter 1 a score : augmente la valeur de 1.

score : lit la valeur actuelle.

Remarque

On peut utiliser les variables dans des calculs. Par exemple, si cote contient 5, alors le bloc cote * cote donne 25 (l'aire du carre).

III. Boucles

Definition

Une boucle permet de repeter un bloc d'instructions plusieurs fois.

repeter 10 fois : execute les instructions a l'interieur 10 fois.

Exemple

Pour dessiner un carre de cote 100 :

repeter 4 fois

avancer de 100

tourner de 90 degres

IV. Conditions

Definition

Une condition permet d'executer des instructions uniquement si un test est verifie.

si ... alors : execute le bloc interieur seulement si la condition est vraie.

si ... alors ... sinon : execute un bloc si la condition est vraie, un autre si elle est fausse.

V. Exercices

Exercice 1

On execute le programme suivant : mettre x a 5, puis ajouter 3 a x, puis mettre x a x * 2. Quelle est la valeur finale de x ?

Exercice 2

Ecrire un programme Scratch qui demande un nombre a l'utilisateur et affiche "Pair" s'il est pair, "Impair" sinon.

Exercice 3

On execute une boucle "repeter 5 fois : ajouter 4 a somme" avec somme initialisee a 0. Quelle est la valeur finale de somme ?

Corriges

Corriges

Ex. 1. $x = 5$, puis $x = 5 + 3 = 8$, puis $x = 8 \times 2 = 16$. Valeur finale : $x = 16$.

Ex. 2. demander "Quel nombre ?". si (reponse modulo 2 = 0) alors dire "Pair" sinon dire "Impair".

Ex. 3. $\text{somme} = 0 + 4 + 4 + 4 + 4 + 4 = 20$.
