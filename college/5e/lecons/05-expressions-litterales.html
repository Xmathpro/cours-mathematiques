Expressions litterales

Ecrire, simplifier et calculer avec des lettres

I. Qu'est-ce qu'une expression litterale ?

Definition

Une expression litterale est une expression mathematique qui contient une ou plusieurs lettres representant des nombres.

Exemples : $3x + 5$, $2a - b$, $x^2 + 1$.

II. Conventions d'ecriture

III. Tester une egalite

Methode

Pour tester si une egalite est vraie pour une valeur donnee, on remplace la lettre par cette valeur dans chaque membre et on compare les resultats.

Exemple

Est-ce que $2x + 3 = 11$ est vraie pour $x = 4$ ?

Membre de gauche : $2 \times 4 + 3 = 8 + 3 = 11$.

Membre de droite : $11$.

$11 = 11$ : l'egalite est vraie pour $x = 4$.

IV. Produire une expression

Exemple : perimetre d'un rectangle

Un rectangle a pour longueur $x$ et pour largeur $3$.

Son perimetre est : $P = 2 \times (x + 3) = 2x + 6$.

Si $x = 7$ cm, alors $P = 2 \times 7 + 6 = 20$ cm.

V. Exercices

Exercice 1 : simplifier les ecritures

Reecris sans le signe $\times$ : a) $4 \times a$ ; b) $a \times b$ ; c) $1 \times x$ ; d) $a \times a$ ; e) $2 \times x \times 3$.

Exercice 2 : calculer

Calcule pour $x = 3$ et $x = -2$ : a) $5x + 1$ ; b) $x^2 - 4$ ; c) $3x - 2x + 7$.

Exercice 3 : tester une egalite

L'egalite $4x - 1 = 19$ est-elle vraie pour $x = 5$ ? Pour $x = 3$ ?

Exercice 4 : probleme

Un triangle equilateral a pour cote $c$ cm. Exprime son perimetre $P$ en fonction de $c$. Calcule $P$ pour $c = 4{,}5$ cm.

Corriges

Corriges

Ex. 1. a) $4a$. b) $ab$. c) $x$. d) $a^2$. e) $6x$.

Ex. 2. Pour $x=3$ : a) $16$, b) $5$, c) $10$. Pour $x=-2$ : a) $-9$, b) $0$, c) $5$.

Ex. 3. $x=5$ : $4 \times 5 - 1 = 19$, vrai. $x=3$ : $4 \times 3 - 1 = 11 \neq 19$, faux.

Ex. 4. $P = 3c$. Pour $c = 4{,}5$ : $P = 3 \times 4{,}5 = 13{,}5$ cm.
