Corrige : Priorites operatoires

Exercice 1

Enonce : Calculer en respectant les priorites : a) $3+5\times 2$ b) $8-4\div 2$ c) $7\times 3+1$.

Reponse : a) $3+10=13$. b) $8-2=6$. c) $21+1=22$.

Exercice 2

Enonce : Calculer : a) $(3+5)\times 2$ b) $(8-4)\div 2$ c) $7\times(3+1)$.

Reponse : a) $8\times 2=16$. b) $4\div 2=2$. c) $7\times 4=28$.

Exercice 3

Enonce : Calculer : $4+3\times(7-2)$.

Reponse : $=4+3\times 5=4+15=19$.

Exercice 4

Enonce : Calculer : $5\times 2+3\times 4-6\div 3$.

Reponse : $=10+12-2=20$.

Exercice 5

Enonce : Calculer : $(12-4\times 2)+5$.

Reponse : $=(12-8)+5=4+5=9$.

Exercice 6

Enonce : Calculer : $[3+(4\times 2)]\times 5$.

Reponse : $=[3+8]\times 5=11\times 5=55$.

Exercice 7

Enonce : Placer des parentheses pour que le calcul $2+3\times 4=20$ soit vrai.

Reponse : $(2+3)\times 4=5\times 4=20$.

Exercice 8

Enonce : Placer des parentheses pour que $8-2\times 3+1=19$ soit vrai.

Reponse : $(8-2)\times(3+1)=6\times 4=24$. Pas de solution pour 19. Autre possibilite : $8-(2\times 3)+1=8-6+1=3$.
