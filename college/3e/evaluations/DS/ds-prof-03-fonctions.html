DS03 : Fonctions (L8-L10)

Devoir surveille (55 min)

Exercice 1 : Notion de fonction (4 pts)

Soit $f(x) = -x^2 + 4x - 3$.

1. (2 pts) Calculer $f(0)$, $f(1)$, $f(2)$, $f(3)$, $f(4)$.

2. (1 pt) Quel est le maximum de $f$ sur $[0 ; 4]$ ? Pour quelle valeur de $x$ ?

3. (1 pt) Resoudre $f(x) = 0$.

Corrige

1. $f(0) = -3$, $f(1) = 0$, $f(2) = 1$, $f(3) = 0$, $f(4) = -3$.

2. Maximum : $f(2) = 1$, atteint pour $x = 2$.

3. $f(x) = 0$ pour $x = 1$ et $x = 3$.

Exercice 2 : Fonction lineaire (4 pts)

1. (2 pts) La fonction lineaire $g$ verifie $g(4) = -6$. Determiner $g(x)$.

2. (1 pt) Calculer $g(-10)$.

3. (1 pt) Determiner l'antecedent de 9 par $g$.

Corrige

1. $g(x) = ax$, $4a = -6$, $a = -\dfrac{3}{2}$. $g(x) = -\dfrac{3}{2}x$.

2. $g(-10) = 15$.

3. $-\dfrac{3}{2}x = 9$, $x = -6$.

Exercice 3 : Fonction affine (5 pts)

1. (2 pts) Determiner la fonction affine $h$ telle que $h(-1) = 5$ et $h(3) = -3$.

2. (1 pt) Tracer la droite representative de $h$.

3. (2 pts) Determiner l'intersection de cette droite avec la droite $y = x + 1$.

Corrige

1. $a = \dfrac{-3 - 5}{3 - (-1)} = \dfrac{-8}{4} = -2$. $h(x) = -2x + b$. $h(-1) = 5$ : $2 + b = 5$, $b = 3$. $h(x) = -2x + 3$.

2. Passe par $(0, 3)$ et $(1, 1)$.

3. $-2x + 3 = x + 1$. $-3x = -2$. $x = \dfrac{2}{3}$. $y = \dfrac{5}{3}$. Point : $\left(\dfrac{2}{3} ; \dfrac{5}{3}\right)$.

Exercice 4 : Probleme (7 pts)

Deux entreprises proposent des tarifs de location de voiture.

Entreprise A : 0,60 euro/km. Entreprise B : forfait de 50 euros + 0,20 euro/km.

1. (2 pts) Exprimer le cout $C_A(x)$ et $C_B(x)$ en fonction du nombre de km $x$.

2. (1 pt) Quelle est la nature de chaque fonction ?

3. (2 pts) Determiner pour quel kilometrage les deux tarifs sont egaux.

4. (2 pts) A partir de quel kilometrage l'entreprise B est-elle plus avantageuse ?

Corrige

1. $C_A(x) = 0{,}6x$ et $C_B(x) = 0{,}2x + 50$.

2. $C_A$ est lineaire, $C_B$ est affine.

3. $0{,}6x = 0{,}2x + 50$. $0{,}4x = 50$. $x = 125$ km.

4. $0{,}2x + 50 < 0{,}6x$. $50 < 0{,}4x$. $x > 125$. B est avantageuse a partir de 126 km.
