DS02 : Identites remarquables, equations et inequations (L4-L7)

Devoir surveille (55 min)

Nom : _________________________ Prenom : _________________________ Classe : _________ Note : ___ / 20

Exercice 1 : Developper (4 pts)

Developper et reduire :

a) $(4x - 3)^2$ ; b) $(2x + 5)(2x - 5)$ ; c) $(x + \sqrt{3})^2$.

Exercice 2 : Factoriser (5 pts)

Factoriser :

a) $9x^2 - 24x + 16$ ; b) $49 - x^2$ ; c) $(3x + 1)^2 - (x - 5)^2$.

Exercice 3 : Equations (5 pts)

Resoudre :

a) $(2x - 1)(x + 4) = 0$ ; b) $9x^2 - 25 = 0$ ; c) $\dfrac{3x + 2}{5} = \dfrac{x - 1}{2}$.

Exercice 4 : Inequations (3 pts)

a) Resoudre $4x - 7 > 2x + 5$.

b) Resoudre $-3x + 12 \leq 0$.

c) Trouver les entiers $n$ tels que $2n + 1 < 15$ et $n > 3$.

Exercice 5 : Probleme (3 pts)

Un rectangle a pour longueur $(x + 5)$ et pour largeur $(x - 2)$ (en cm). Pour quelles valeurs de $x$ le perimetre est-il superieur a 30 cm ?
