IC03 : Identites remarquables (L4-L5)

Interrogation de cours (20 min)

Question

1. (3 pts) Developper : a) $(3x+2)^2$ ; b) $(x-7)^2$ ; c) $(4x+3)(4x-3)$.

Reponse

a) $9x^2 + 12x + 4$. b) $x^2 - 14x + 49$. c) $16x^2 - 9$.

Question

2. (3 pts) Factoriser : a) $x^2 + 6x + 9$ ; b) $25x^2 - 36$ ; c) $4x^2 - 20x + 25$.

Reponse

a) $(x+3)^2$. b) $(5x+6)(5x-6)$. c) $(2x-5)^2$.

Question

3. (2 pts) Calculer $103^2$ mentalement.

Reponse

$(100+3)^2 = 10\,000 + 600 + 9 = 10\,609$.

Question

4. (2 pts) Factoriser $(x+1)^2 - 9$.

Reponse

$(x+1)^2 - 3^2 = (x+1+3)(x+1-3) = (x+4)(x-2)$.
