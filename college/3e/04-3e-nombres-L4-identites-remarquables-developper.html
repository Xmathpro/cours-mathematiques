Identites remarquables : developper

Carre d'une somme, carre d'une difference, produit somme-difference

I. Carre d'une somme

Propriete

Pour tous nombres $a$ et $b$ : $(a + b)^2 = a^2 + 2ab + b^2$.

Preuve

$(a + b)^2 = (a + b)(a + b) = a \times a + a \times b + b \times a + b \times b = a^2 + ab + ab + b^2 = a^2 + 2ab + b^2$.

Interpretation geometrique : (a + b)^2

II. Carre d'une difference

Propriete

Pour tous nombres $a$ et $b$ : $(a - b)^2 = a^2 - 2ab + b^2$.

Exemples de developpement

$(x + 5)^2 = x^2 + 2 \times x \times 5 + 5^2 = x^2 + 10x + 25$.

$(3x - 4)^2 = (3x)^2 - 2 \times 3x \times 4 + 4^2 = 9x^2 - 24x + 16$.

III. Produit somme-difference

Propriete

Pour tous nombres $a$ et $b$ : $(a + b)(a - b) = a^2 - b^2$.

Exemples

$(x + 7)(x - 7) = x^2 - 49$.

$(2x + 3)(2x - 3) = 4x^2 - 9$.

IV. Applications au calcul mental

Exemples

$101^2 = (100 + 1)^2 = 10000 + 200 + 1 = 10201$.

$99^2 = (100 - 1)^2 = 10000 - 200 + 1 = 9801$.

$52 \times 48 = (50 + 2)(50 - 2) = 2500 - 4 = 2496$.

Exercices

Exercice 1

Developper et reduire : a) $(x + 3)^2$ ; b) $(2x - 1)^2$ ; c) $(x + 4)(x - 4)$ ; d) $(5x + 2)^2$.

Corrige

a) $(x + 3)^2 = x^2 + 6x + 9$.

b) $(2x - 1)^2 = 4x^2 - 4x + 1$.

c) $(x + 4)(x - 4) = x^2 - 16$.

d) $(5x + 2)^2 = 25x^2 + 20x + 4$.

Exercice 2

Developper et reduire : a) $(3x + 2y)^2$ ; b) $(x - 5)(x + 5) + (x + 3)^2$.

Corrige

a) $(3x + 2y)^2 = 9x^2 + 12xy + 4y^2$.

b) $(x - 5)(x + 5) + (x + 3)^2 = x^2 - 25 + x^2 + 6x + 9 = 2x^2 + 6x - 16$.

Exercice 3

Calculer mentalement en utilisant une identite remarquable : a) $103^2$ ; b) $97^2$ ; c) $31 \times 29$.

Corrige

a) $103^2 = (100 + 3)^2 = 10000 + 600 + 9 = 10609$.

b) $97^2 = (100 - 3)^2 = 10000 - 600 + 9 = 9409$.

c) $31 \times 29 = (30 + 1)(30 - 1) = 900 - 1 = 899$.

Exercice 4

Developper et reduire : $(2x + 3)^2 - (2x - 3)^2$.

Corrige

$(2x + 3)^2 = 4x^2 + 12x + 9$ et $(2x - 3)^2 = 4x^2 - 12x + 9$.

$(2x + 3)^2 - (2x - 3)^2 = 4x^2 + 12x + 9 - 4x^2 + 12x - 9 = 24x$.

Exercice 5

Montrer que pour tout nombre $x$ : $(x + 1)^2 - (x - 1)^2 = 4x$.

Corrige

$(x + 1)^2 - (x - 1)^2 = (x^2 + 2x + 1) - (x^2 - 2x + 1) = x^2 + 2x + 1 - x^2 + 2x - 1 = 4x$.
