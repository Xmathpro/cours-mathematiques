Fonctions lineaires

Definition, coefficient, proportionnalite, representation graphique

I. Definition

Definition

Une fonction lineaire est une fonction de la forme $f : x \mapsto ax$ ou $a$ est un nombre fixe appele coefficient de la fonction lineaire.

Propriete

Une situation de proportionnalite est modelisee par une fonction lineaire. Le coefficient $a$ est le coefficient de proportionnalite.

Exemple

Le prix d'un kilogramme de pommes est 2,50 euros. Le prix de $x$ kg est $f(x) = 2{,}5x$. C'est une fonction lineaire de coefficient 2,5.

II. Representation graphique

Propriete

La representation graphique d'une fonction lineaire $f : x \mapsto ax$ est une droite passant par l'origine du repere. Le nombre $a$ est le coefficient directeur de cette droite.

III. Determiner le coefficient

Propriete

Si $f$ est lineaire et $f(x_0) = y_0$ avec $x_0 \neq 0$, alors $a = \dfrac{y_0}{x_0} = \dfrac{f(x_0)}{x_0}$.

Exercices

Exercice 1

Soit $f : x \mapsto -3x$. Calculer $f(2)$, $f(-4)$ et $f\left(\dfrac{1}{3}\right)$.

Corrige

$f(2) = -6$. $f(-4) = 12$. $f\left(\dfrac{1}{3}\right) = -1$.

Exercice 2

$g$ est une fonction lineaire telle que $g(4) = 10$. Determiner le coefficient de $g$ puis exprimer $g(x)$.

Corrige

$a = \dfrac{g(4)}{4} = \dfrac{10}{4} = 2{,}5$. Donc $g(x) = 2{,}5x$.

Exercice 3

Le prix d'un abonnement telephonique est proportionnel a la duree. Un abonnement de 3 mois coute 45 euros. a) Exprimer le prix $P$ en fonction de la duree $d$ en mois. b) Calculer le prix pour 7 mois.

Corrige

a) $P(d) = 15d$ (coefficient : $45 \div 3 = 15$).

b) $P(7) = 15 \times 7 = 105$ euros.

Exercice 4

Parmi les fonctions suivantes, lesquelles sont lineaires ? a) $f(x) = 3x$ ; b) $g(x) = 2x + 1$ ; c) $h(x) = -x$ ; d) $k(x) = x^2$.

Corrige

a) $f$ est lineaire (coefficient 3). c) $h$ est lineaire (coefficient $-1$).

b) $g$ n'est pas lineaire (c'est une fonction affine). d) $k$ n'est pas lineaire.
