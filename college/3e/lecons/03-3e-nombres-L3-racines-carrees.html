Racines carrees

Definition, proprietes, simplification, operations

I. Definition

Definition

Soit $a$ un nombre positif. La racine carree de $a$, notee $\sqrt{a}$, est le nombre positif dont le carre est egal a $a$.

Autrement dit : $\sqrt{a} \geq 0$ et $\left(\sqrt{a}\right)^2 = a$.

Remarques

$\sqrt{a}$ n'existe que si $a \geq 0$. On ne peut pas calculer la racine carree d'un nombre negatif.

$\sqrt{0} = 0$, $\sqrt{1} = 1$, $\sqrt{4} = 2$, $\sqrt{9} = 3$, $\sqrt{16} = 4$, $\sqrt{25} = 5$.

II. Proprietes

Propriete : produit

Pour tous nombres positifs $a$ et $b$ : $\sqrt{a \times b} = \sqrt{a} \times \sqrt{b}$.

Propriete : quotient

Pour $a \geq 0$ et $b > 0$ : $\sqrt{\dfrac{a}{b}} = \dfrac{\sqrt{a}}{\sqrt{b}}$.

Attention

Il n'existe aucune formule pour $\sqrt{a + b}$. En general, $\sqrt{a+b} \neq \sqrt{a} + \sqrt{b}$.

Exemple : $\sqrt{9 + 16} = \sqrt{25} = 5$ mais $\sqrt{9} + \sqrt{16} = 3 + 4 = 7$.

III. Simplification de racines carrees

Methode

Pour simplifier $\sqrt{n}$, on cherche le plus grand carre parfait qui divise $n$.

$\sqrt{n} = \sqrt{k^2 \times m} = k\sqrt{m}$ ou $m$ ne contient plus de facteur carre.

Exemples

$\sqrt{50} = \sqrt{25 \times 2} = \sqrt{25} \times \sqrt{2} = 5\sqrt{2}$.

$\sqrt{72} = \sqrt{36 \times 2} = 6\sqrt{2}$.

$\sqrt{48} = \sqrt{16 \times 3} = 4\sqrt{3}$.

$\sqrt{200} = \sqrt{100 \times 2} = 10\sqrt{2}$.

IV. Operations avec des racines carrees

Propriete

On peut additionner ou soustraire des racines carrees uniquement si elles ont le meme radicande (le meme nombre sous la racine).

$a\sqrt{n} + b\sqrt{n} = (a+b)\sqrt{n}$.

Exemple

$3\sqrt{5} + 7\sqrt{5} = 10\sqrt{5}$.

$\sqrt{12} + \sqrt{27} = 2\sqrt{3} + 3\sqrt{3} = 5\sqrt{3}$.

Exercices

Exercice 1

Simplifier : a) $\sqrt{75}$ ; b) $\sqrt{98}$ ; c) $\sqrt{128}$ ; d) $\sqrt{180}$.

Corrige

a) $\sqrt{75} = \sqrt{25 \times 3} = 5\sqrt{3}$.

b) $\sqrt{98} = \sqrt{49 \times 2} = 7\sqrt{2}$.

c) $\sqrt{128} = \sqrt{64 \times 2} = 8\sqrt{2}$.

d) $\sqrt{180} = \sqrt{36 \times 5} = 6\sqrt{5}$.

Exercice 2

Calculer : a) $\sqrt{3} \times \sqrt{12}$ ; b) $\dfrac{\sqrt{50}}{\sqrt{2}}$ ; c) $\sqrt{5} \times \sqrt{20}$.

Corrige

a) $\sqrt{3} \times \sqrt{12} = \sqrt{3 \times 12} = \sqrt{36} = 6$.

b) $\dfrac{\sqrt{50}}{\sqrt{2}} = \sqrt{\dfrac{50}{2}} = \sqrt{25} = 5$.

c) $\sqrt{5} \times \sqrt{20} = \sqrt{100} = 10$.

Exercice 3

Ecrire sous la forme $a\sqrt{b}$ avec $b$ le plus petit possible : $\sqrt{45} + \sqrt{20} - \sqrt{80}$.

Corrige

$\sqrt{45} = 3\sqrt{5}$, $\sqrt{20} = 2\sqrt{5}$, $\sqrt{80} = 4\sqrt{5}$.

$3\sqrt{5} + 2\sqrt{5} - 4\sqrt{5} = (3 + 2 - 4)\sqrt{5} = \sqrt{5}$.

Exercice 4

Developper et simplifier : $(3 + \sqrt{2})^2$.

Corrige

$(3 + \sqrt{2})^2 = 3^2 + 2 \times 3 \times \sqrt{2} + (\sqrt{2})^2 = 9 + 6\sqrt{2} + 2 = 11 + 6\sqrt{2}$.

Exercice 5

Un carre a pour aire 288 cm$^2$. Exprimer la longueur de son cote sous la forme $a\sqrt{b}$ avec $b$ entier le plus petit possible.

Corrige

Le cote mesure $\sqrt{288}$ cm. $288 = 144 \times 2$. Donc $\sqrt{288} = \sqrt{144 \times 2} = 12\sqrt{2}$ cm.
