Identites remarquables : factoriser

Reconnaitre et utiliser les identites dans le sens de la factorisation

I. Rappel des trois identites

Propriete : identites remarquables (sens factorisation)

$a^2 + 2ab + b^2 = (a + b)^2$

$a^2 - 2ab + b^2 = (a - b)^2$

$a^2 - b^2 = (a + b)(a - b)$

II. Factoriser avec $a^2 - b^2$

Methode

On repere une difference de deux carres : on identifie $a$ et $b$ tels que l'expression s'ecrive $a^2 - b^2$, puis on factorise en $(a + b)(a - b)$.

Exemples

$x^2 - 9 = x^2 - 3^2 = (x + 3)(x - 3)$.

$4x^2 - 25 = (2x)^2 - 5^2 = (2x + 5)(2x - 5)$.

$16x^2 - 49y^2 = (4x)^2 - (7y)^2 = (4x + 7y)(4x - 7y)$.

III. Factoriser avec $(a + b)^2$ ou $(a - b)^2$

Methode

On repere un trinome carre parfait : trois termes dont le premier et le dernier sont des carres parfaits et le terme du milieu est le double produit.

Exemples

$x^2 + 6x + 9 = x^2 + 2 \times x \times 3 + 3^2 = (x + 3)^2$.

$9x^2 - 30x + 25 = (3x)^2 - 2 \times 3x \times 5 + 5^2 = (3x - 5)^2$.

Entrainement : reconnaitre l'identite

IV. Factorisation en plusieurs etapes

Methode

Parfois, on doit d'abord mettre un facteur commun en evidence, puis appliquer une identite remarquable.

Exemple

Factoriser $3x^2 - 27$.

$3x^2 - 27 = 3(x^2 - 9) = 3(x + 3)(x - 3)$.

Exercices

Exercice 1

Factoriser : a) $x^2 - 81$ ; b) $x^2 + 12x + 36$ ; c) $4x^2 - 1$ ; d) $x^2 - 22x + 121$.

Corrige

a) $x^2 - 81 = (x + 9)(x - 9)$.

b) $x^2 + 12x + 36 = (x + 6)^2$.

c) $4x^2 - 1 = (2x + 1)(2x - 1)$.

d) $x^2 - 22x + 121 = (x - 11)^2$.

Exercice 2

Factoriser : a) $2x^2 - 50$ ; b) $5x^2 + 30x + 45$ ; c) $12x^2 - 75$.

Corrige

a) $2x^2 - 50 = 2(x^2 - 25) = 2(x + 5)(x - 5)$.

b) $5x^2 + 30x + 45 = 5(x^2 + 6x + 9) = 5(x + 3)^2$.

c) $12x^2 - 75 = 3(4x^2 - 25) = 3(2x + 5)(2x - 5)$.

Exercice 3

Factoriser $(x + 3)^2 - (2x - 1)^2$.

Corrige

On pose $A = (x + 3)$ et $B = (2x - 1)$. C'est de la forme $A^2 - B^2 = (A + B)(A - B)$.

$A + B = (x + 3) + (2x - 1) = 3x + 2$.

$A - B = (x + 3) - (2x - 1) = x + 3 - 2x + 1 = -x + 4$.

$(x + 3)^2 - (2x - 1)^2 = (3x + 2)(-x + 4)$.

Exercice 4

Montrer que $997^2 - 3^2$ est divisible par 1000.

Corrige

$997^2 - 3^2 = (997 + 3)(997 - 3) = 1000 \times 994$.

Donc $997^2 - 3^2 = 1000 \times 994$ est bien divisible par 1000.

Exercice 5

Factoriser $A = (3x + 1)^2 - (x - 5)^2$ puis resoudre $A = 0$.

Corrige

$A = [(3x + 1) + (x - 5)][(3x + 1) - (x - 5)] = (4x - 4)(2x + 6) = 2(2x - 2)(2x + 6)$.

On peut simplifier : $A = (4x - 4)(2x + 6) = 2(2x - 2) \times 2(x + 3) = 4(2x - 2)(x + 3)$.

$A = 0$ quand $4x - 4 = 0$ ou $2x + 6 = 0$, soit $x = 1$ ou $x = -3$.
