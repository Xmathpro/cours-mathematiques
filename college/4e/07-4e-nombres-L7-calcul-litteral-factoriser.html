Calcul litteral : factoriser

Facteur commun, factorisation

I. Factoriser avec un facteur commun

Definition. Factoriser une expression, c'est la transformer en un produit. C'est l'operation inverse du developpement.

Propriete. Si une expression est de la forme $ka + kb$, on peut factoriser par $k$ :

$$ka + kb = k(a + b)$$

Methode. Pour factoriser, on identifie le facteur commun a tous les termes, puis on le met en evidence.

Exemples.

  • $6x + 15 = 3(2x + 5)$ (facteur commun : 3)
  • $x^2 + 4x = x(x + 4)$ (facteur commun : $x$)
  • $12x^2 - 8x = 4x(3x - 2)$ (facteur commun : $4x$)

II. Reconnaitre une factorisation

Methode. Pour verifier une factorisation, il suffit de developper le resultat et de retrouver l'expression de depart.

Exemple. Verifions que $6x + 15 = 3(2x + 5)$ : on developpe $3(2x + 5) = 6x + 15$. C'est correct.

Exercices

Exercice 1. Factoriser les expressions suivantes.

a) $8x + 12$    b) $5x - 20$    c) $x^2 + 7x$    d) $6x^2 - 9x$

Correction.

a) $8x + 12 = 4(2x + 3)$.

b) $5x - 20 = 5(x - 4)$.

c) $x^2 + 7x = x(x + 7)$.

d) $6x^2 - 9x = 3x(2x - 3)$.

Exercice 2. Factoriser $A = 15x^2 + 10x$.

Correction. $A = 5x(3x + 2)$.

Exercice 3. Factoriser $B = (x + 3)(2x) + (x + 3)(5)$.

Correction. Le facteur commun est $(x + 3)$ : $B = (x + 3)(2x + 5)$.

Exercice 4. Developper puis factoriser $3(x + 2) + x(x + 2)$.

Correction. Facteur commun $(x + 2)$ : $3(x + 2) + x(x + 2) = (x + 2)(3 + x) = (x + 2)(x + 3)$.

Verification par developpement : $(x + 2)(x + 3) = x^2 + 3x + 2x + 6 = x^2 + 5x + 6$. Et $3(x + 2) + x(x + 2) = 3x + 6 + x^2 + 2x = x^2 + 5x + 6$. C'est correct.

Exercice 5. Factoriser au maximum $C = 18x^3 - 12x^2 + 6x$.

Correction. $C = 6x(3x^2 - 2x + 1)$.
