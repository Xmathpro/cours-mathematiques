Exercices : Calcul litteral : factoriser

Corrige

Exercice 1. (2 points) Factoriser : a) $8x + 12$ b) $5x - 20$ c) $x^2 + 7x$ d) $6x^2 - 9x$.

Correction. a) $4(2x+3)$ b) $5(x-4)$ c) $x(x+7)$ d) $3x(2x-3)$.

Exercice 2. (2 points) Factoriser $A = 15x^2 + 10x$.

Correction. $A = 5x(3x + 2)$.

Exercice 3. (2 points) Factoriser $B = (x+3)(2x) + (x+3)(5)$.

Correction. Facteur commun $(x+3)$ : $B = (x+3)(2x+5)$.

Exercice 4. (3 points) Factoriser $3(x+2) + x(x+2)$.

Correction. Facteur commun $(x+2)$ : $(x+2)(3+x) = (x+2)(x+3)$.

Exercice 5. (3 points) Factoriser au maximum $C = 18x^3 - 12x^2 + 6x$.

Correction. $C = 6x(3x^2 - 2x + 1)$.

Exercice 6. (3 points) Factoriser $D = (2x-1)(x+4) + (2x-1)(3x-2)$.

Correction. Facteur commun $(2x-1)$ : $D = (2x-1)(x+4+3x-2) = (2x-1)(4x+2) = 2(2x-1)(2x+1)$.
