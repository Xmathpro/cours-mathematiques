Calcul litteral : developper

Simple distributivite, double distributivite

I. Simple distributivite

Propriete. Pour tous nombres $k$, $a$ et $b$ :

$$k(a + b) = ka + kb \qquad k(a - b) = ka - kb$$

Exemples.

  • $3(x + 5) = 3x + 15$
  • $-2(4x - 7) = -8x + 14$ (attention au signe moins devant le 2)
  • $x(x + 3) = x^2 + 3x$

II. Double distributivite

Propriete. Pour tous nombres $a$, $b$, $c$ et $d$ :

$$(a + b)(c + d) = ac + ad + bc + bd$$

Exemples.

$(x + 3)(x + 5) = x \times x + x \times 5 + 3 \times x + 3 \times 5 = x^2 + 5x + 3x + 15 = x^2 + 8x + 15$

$(2x - 1)(x + 4) = 2x^2 + 8x - x - 4 = 2x^2 + 7x - 4$

III. Reduire une expression

Methode. Apres avoir developpe, on regroupe les termes de meme nature (les termes en $x^2$, les termes en $x$, les constantes).

Exercices

Exercice 1. Developper et reduire.

a) $5(2x + 3)$    b) $-3(x - 4)$    c) $x(2x - 7)$

Correction.

a) $5(2x + 3) = 10x + 15$.

b) $-3(x - 4) = -3x + 12$.

c) $x(2x - 7) = 2x^2 - 7x$.

Exercice 2. Developper et reduire $(x + 2)(x + 6)$.

Correction. $(x + 2)(x + 6) = x^2 + 6x + 2x + 12 = x^2 + 8x + 12$.

Exercice 3. Developper et reduire $(3x - 5)(2x + 1)$.

Correction. $(3x - 5)(2x + 1) = 6x^2 + 3x - 10x - 5 = 6x^2 - 7x - 5$.

Exercice 4. Developper et reduire $A = 3(x + 4) - 2(x - 1)$.

Correction. $A = 3x + 12 - 2x + 2 = x + 14$.

Exercice 5. Montrer que $(x + 3)^2 = x^2 + 6x + 9$.

Correction. $(x + 3)^2 = (x + 3)(x + 3) = x^2 + 3x + 3x + 9 = x^2 + 6x + 9$.
