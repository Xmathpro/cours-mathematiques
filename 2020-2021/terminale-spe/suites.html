Suites numériques

Limites de suites, récurrence, suites auxiliaires et Python

I. Limites de suites

Limites de suites géométriques $q^n$
  • $|q|<1$ : $\lim q^n = 0$
  • $q=1$ : $\lim q^n = 1$
  • $q>1$ : $\lim q^n = +\infty$
  • $q \leq -1$ : pas de limite
Théorème des gendarmes

Si $u_n \leq v_n \leq w_n$ et $\lim u_n = \lim w_n = \ell$, alors $\lim v_n = \ell$.

Convergence monotone

Toute suite croissante majorée converge. Toute suite décroissante minorée converge.

II. Raisonnement par récurrence

Méthode

Initialisation : vérifier $P(n_0)$.

Hérédité : supposer $P(n)$ et montrer $P(n+1)$.

Conclusion : $P(n)$ vraie pour tout $n \geq n_0$.

Explorer : convergence d'une suite récurrente $u_{n+1} = au_n + b$

III. Suites auxiliaires

Suite $u_{n+1} = au_n + b$ ($a \neq 1$)

Point fixe : $\ell = \dfrac{b}{1-a}$. On pose $v_n = u_n - \ell$. Alors $(v_n)$ est géométrique de raison $a$ :

$$u_n = (u_0 - \ell) \cdot a^n + \ell$$

IV. Python et suites

Calculer les termes d'une suite récurrente
def suite_recursive(u0, a, b, n):
    u = u0
    for i in range(n):
        u = a * u + b
    return u

# u0 = 10, u_{n+1} = 0.5*u_n + 2
for n in range(20):
    print(f"u_{n} = {suite_recursive(10, 0.5, 2, n):.4f}")
# Converge vers 4 (point fixe : 2/(1-0.5) = 4)

V. Exercices

Exercice 1

$u_0 = 8$ et $u_{n+1} = 0{,}7u_n + 3$.

  1. Calculer le point fixe.
  2. Exprimer $u_n$ en fonction de $n$.
  3. Calculer $\lim u_n$.

Exercice 2

Montrer par récurrence que $2^n \geq n^2$ pour tout $n \geq 4$.

VI. Corrigés

Corrigé exercice 1

Corrigé

1. $\ell = \dfrac{3}{1-0{,}7} = \dfrac{3}{0{,}3} = 10$.

2. $u_n = (8-10) \times 0{,}7^n + 10 = -2 \times 0{,}7^n + 10$.

3. $|0{,}7| < 1$ donc $0{,}7^n \to 0$. $\lim u_n = 10$.

Corrigé exercice 2

Corrigé

Init ($n=4$) : $2^4 = 16 \geq 4^2 = 16$. Vrai.

Hérédité : supposons $2^n \geq n^2$ pour $n \geq 4$.

$2^{n+1} = 2 \times 2^n \geq 2n^2$. Il suffit de montrer $2n^2 \geq (n+1)^2 = n^2+2n+1$.

$2n^2 \geq n^2+2n+1 \iff n^2-2n-1 \geq 0 \iff (n-1)^2 \geq 2$. Vrai pour $n \geq 4$.
