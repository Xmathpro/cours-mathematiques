SQL : interroger une base de données

SELECT, WHERE, JOIN, agrégation et modèle relationnel

I. Le modèle relationnel

Définitions

Une base de données relationnelle organise les données en tables (ou relations). Chaque table a :

  • Des colonnes (ou attributs) : le nom, le type de chaque donnée.
  • Des lignes (ou enregistrements, ou tuples) : chaque ligne est une entrée.
  • Une clé primaire : un attribut unique qui identifie chaque ligne.
  • Des clés étrangères : des attributs qui font référence à une autre table.

II. SELECT : lire des données

-- Sélectionner toutes les colonnes
SELECT *
FROM eleves;

-- Sélectionner certaines colonnes
SELECT prenom, nom
FROM eleves;

-- Filtrer avec WHERE
SELECT prenom, nom, age
FROM eleves
WHERE age >= 17;

-- Trier les résultats
SELECT prenom, nom, age
FROM eleves
ORDER BY age DESC;

III. Fonctions d'agrégation

-- Compter le nombre d'élèves
SELECT COUNT(*) AS nombre
FROM eleves;

-- Calculer la moyenne d'âge
SELECT AVG(age) AS age_moyen
FROM eleves;

-- Grouper par classe
SELECT classe_id, COUNT(*) AS effectif
FROM eleves
GROUP BY classe_id;

-- Filtrer les groupes avec HAVING
SELECT classe_id, COUNT(*) AS effectif
FROM eleves
GROUP BY classe_id
HAVING COUNT(*) >= 2;

IV. JOIN : croiser des tables

La jointure

Un JOIN permet de combiner les données de deux tables en utilisant une clé commune (la clé étrangère).

-- Table classes
-- id | nom         | professeur
-- 1  | 1ère NSI A  | M. Vawda
-- 2  | 1ère NSI B  | Mme Dupont
-- 3  | Term NSI    | M. Vawda

-- Afficher le prénom de l'élève ET le nom de sa classe
SELECT eleves.prenom,
       eleves.nom,
       classes.nom AS classe
FROM eleves
JOIN classes ON eleves.classe_id = classes.id;

-- Résultat :
-- Alice  | Martin | 1ère NSI A
-- Bob    | Dupont | 1ère NSI B
-- Clara  | Leroy  | 1ère NSI A
-- David  | Moreau | Term NSI
-- Emma   | Garcia | 1ère NSI B

Visualiser : comment fonctionne un JOIN

Le JOIN relie chaque ligne de la table "eleves" à la ligne correspondante de la table "classes" via classe_id = id.

V. INSERT, UPDATE, DELETE

-- Ajouter un élève
INSERT INTO eleves (prenom, nom, age, classe_id)
VALUES ('Fatou', 'Diallo', 16, 1);

-- Modifier l'âge de Bob
UPDATE eleves
SET age = 18
WHERE prenom = 'Bob';

-- Supprimer un élève
DELETE FROM eleves
WHERE id = 4;

VI. Exercices

Exercice 1 : requêtes simples

En utilisant la table "eleves" ci-dessus :

  1. Écrire la requête qui affiche le prénom et l'âge de tous les élèves.
  2. Afficher les élèves de la classe 2, triés par nom.
  3. Compter le nombre d'élèves de 16 ans.

Exercice 2 : jointure

  1. Écrire la requête qui affiche le prénom de l'élève et le nom de son professeur.
  2. Afficher le nombre d'élèves par professeur.

VII. Corrigés

Corrigé exercice 1

Corrigé
-- 1
SELECT prenom, age
FROM eleves;

-- 2
SELECT *
FROM eleves
WHERE classe_id = 2
ORDER BY nom;

-- 3
SELECT COUNT(*)
FROM eleves
WHERE age = 16;
-- Résultat : 2

Corrigé exercice 2

Corrigé
-- 1
SELECT eleves.prenom, classes.professeur
FROM eleves
JOIN classes ON eleves.classe_id = classes.id;

-- 2
SELECT classes.professeur, COUNT(*) AS nb_eleves
FROM eleves
JOIN classes ON eleves.classe_id = classes.id
GROUP BY classes.professeur;

-- Résultat :
-- M. Vawda   | 3
-- Mme Dupont | 2
