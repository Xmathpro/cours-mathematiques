I. La pile (stack)

Définition

Une pile fonctionne comme une pile d'assiettes : on ne peut ajouter ou retirer que par le sommet. C'est le principe LIFO : Last In, First Out (dernier entré, premier sorti).

Deux opérations fondamentales : empiler (push) et depiler (pop).

Visualiser : la pile d'assiettes

LIFO : le dernier empilé sera le premier dépilé.
class Pile:
    def __init__(self):
        self.elements = []

    def empiler(self, valeur):
        """Ajoute un élément au sommet."""
        self.elements.append(valeur)

    def depiler(self):
        """Retire et retourne l'élément du sommet."""
        if self.est_vide():
            raise IndexError("Pile vide")
        return self.elements.pop()

    def sommet(self):
        """Retourne le sommet sans le retirer."""
        return self.elements[-1]

    def est_vide(self):
        return len(self.elements) == 0

    def taille(self):
        return len(self.elements)


# Exemple : vérifier les parenthèses
def parentheses_valides(expression):
    p = Pile()
    for car in expression:
        if car == '(':
            p.empiler(car)
        elif car == ')':
            if p.est_vide():
                return False
            p.depiler()
    return p.est_vide()

print(parentheses_valides("((a+b)*(c-d))"))  # True
print(parentheses_valides("((a+b)"))          # False

II. La file (queue)

Définition

Une file fonctionne comme une file d'attente : le premier arrivé est le premier servi. C'est le principe FIFO : First In, First Out.

Deux opérations : enfiler (enqueue) et defiler (dequeue).

Visualiser : la file d'attente

FIFO : le premier enfilé sera le premier défilé.
class File:
    def __init__(self):
        self.elements = []

    def enfiler(self, valeur):
        """Ajoute un élément à la fin."""
        self.elements.append(valeur)

    def defiler(self):
        """Retire et retourne le premier élément."""
        if self.est_vide():
            raise IndexError("File vide")
        return self.elements.pop(0)

    def est_vide(self):
        return len(self.elements) == 0

III. Le dictionnaire

Définition

Un dictionnaire associe des clés à des valeurs. Chaque clé est unique. L'accès par clé est très rapide : $O(1)$ en moyenne.

# Créer un dictionnaire
eleve = {
    "prenom": "Alice",
    "age": 16,
    "classe": "1ère NSI",
    "notes": [15, 12, 18, 14]
}

# Accéder à une valeur
print(eleve["prenom"])     # Alice
print(eleve["age"])        # 16

# Modifier
eleve["age"] = 17

# Ajouter
eleve["email"] = "alice@mail.com"

# Parcourir
for cle, valeur in eleve.items():
    print(f"{cle} : {valeur}")

Visualiser : le dictionnaire (table de hachage)

Chaque clé est transformée en indice par une fonction de hachage. L'accès est instantané.

IV. La liste chaînée

Définition

Une liste chaînée est une suite de maillons. Chaque maillon contient une valeur et un pointeur vers le maillon suivant. Le dernier maillon pointe vers None.

Contrairement à un tableau, les éléments ne sont pas côte à côte en mémoire.

Visualiser : la liste chaînée

Chaque maillon pointe vers le suivant. L'ajout et la suppression en tête sont en O(1).
class Maillon:
    def __init__(self, valeur, suivant=None):
        self.valeur = valeur
        self.suivant = suivant


class ListeChainee:
    def __init__(self):
        self.tete = None

    def ajouter_en_tete(self, valeur):
        self.tete = Maillon(valeur, self.tete)

    def afficher(self):
        maillon = self.tete
        while maillon is not None:
            print(maillon.valeur, end=" -> ")
            maillon = maillon.suivant
        print("None")


# Exemple
lst = ListeChainee()
lst.ajouter_en_tete("C")
lst.ajouter_en_tete("B")
lst.ajouter_en_tete("A")
lst.afficher()   # A -> B -> C -> None

def inverser_chaine(chaine):
    p = Pile()
    for car in chaine:
        p.empiler(car)

    resultat = ""
    while not p.est_vide():
        resultat += p.depiler()

    return resultat

print(inverser_chaine("bonjour"))  # ruojnob


def evaluer_npi(expression):
    p = Pile()
    for token in expression.split():
        if token in "+-*/":
            b = p.depiler()
            a = p.depiler()
            if token == '+': p.empiler(a + b)
            elif token == '-': p.empiler(a - b)
            elif token == '*': p.empiler(a * b)
            elif token == '/': p.empiler(a / b)
        else:
            p.empiler(float(token))
    return p.depiler()

print(evaluer_npi("3 4 + 2 *"))  # 14.0

def compter_lettres(mot):
    compteur = {}
    for lettre in mot.lower():
        if lettre in compteur:
            compteur[lettre] += 1
        else:
            compteur[lettre] = 1
    return compteur

print(compter_lettres("algorithme"))
# {'a': 1, 'l': 1, 'g': 1, 'o': 1, 'r': 1,
#  'i': 1, 't': 1, 'h': 1, 'm': 1, 'e': 1}

print(compter_lettres("ananas"))
# {'a': 3, 'n': 2, 's': 1}
